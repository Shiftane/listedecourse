Comment ca marche ?
  1. Recherchez une recette à l'aide de la barre de recherche
  2. Choisissez parmi les meilleures recettes celle que vous voulez préparer
  3. Définissez le nombre de personne qui viendra manger avec vous
  4. Ajoutez la recette à votre liste de course
  5. Vous pouvez ensuite ajouter le nombre que vous voulez de recette
  6. Quand vous avez terminé, cliquer sur imprimer dans la fenêtre du bouton recette en haut à droite
